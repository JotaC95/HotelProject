********** OLVIDA EL BACKEND ********
Para el backend

Entra en la carpeta
cd HotelBackend

Crear y activar entorno virtual
python -m venv venv

source venv/bin/activate


Si te pide dependencias 
pip install django djangorestframework

Iniciar el servidor
pip install django djangorestframework
************ IGNORA EL BACKEND *********

Para el front

Entra a la carpeta del front 
cd HotelFlow

Inicia el servidor de Expo 
npx expo start (Solo para la misma red si es una diferente usar un tunel)
